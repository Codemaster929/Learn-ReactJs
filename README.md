## 🚀 Projects

### 📝 Notes App (`13-notes-app`)
A sticky-notes style app that puts together multiple React concepts:
- Component-based UI
- `useState` for managing note data
- Form handling & controlled inputs
- Two-way binding
- Tailwind CSS for styling
- `.map()` to render dynamic lists

---

## 🛠️ Tech Stack

- **React** (with Vite)
- **JavaScript (JSX)**
- **Tailwind CSS**
- **CSS Modules**

---

## ▶️ Running Any Project

1. Navigate into a project folder:
   ```bash
   cd 13-notes-app
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Start the dev server:
   ```bash
   npm run dev
   ```
